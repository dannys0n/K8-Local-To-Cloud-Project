# Kubernetes Development Environment

This project provides a comprehensive Kubernetes development environment with monitoring, management tools, and databases.

## Prerequisites

- kubectl
- Helm
- A running Kubernetes cluster

## Setup

To set up the entire environment, run:

```bash
make up
```

This command will:
1. Set up the cluster (`make cluster`)
2. Install monitoring tools (`make monitors`)
3. Install management tools (`make managers`)
4. Set up databases (`make databases`)
5. Configure port forwarding (`make port-forward`)

### Individual Components

You can also set up components individually:

```bash
# Install monitoring tools (Prometheus + Grafana)
make monitors

# Install management tools (Portainer)
make managers

# Set up databases
make databases

# Configure port forwarding
make port-forward
```

## Services

Once set up, the following services will be available:

### Monitoring
- **Grafana**: http://localhost:3000
- **Prometheus**: http://localhost:9090

### Management
- **Portainer**: http://localhost:9000 (create admin on first visit)

### Databases
- **Redis**: localhost:6379
- **PostgreSQL**: localhost:5432 (credentials: postgres/postgres, database: app)

## Testing Database Connections

Test connections to the databases:

```bash
# Test Redis
make ping-redis

# Test PostgreSQL
make ping-postgres
```

## Status Check

Check the status of monitoring pods:

```bash
make status
```

## Teardown

To clean up and remove all deployed components:

```bash
make down
```

To get the Grafana admin password:

